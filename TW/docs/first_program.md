# 第一個程式
## 文字編輯器
所有程式設計都在程式碼視窗中完成。每個程式碼視窗對應一個包含程式碼的文字檔案。
你可以點擊視窗頂部的檔案名稱來重新命名檔案。

只要程式碼未運行，就可以像在任何文字編輯器中一樣編輯程式碼。
你可以點擊程式碼視窗中的綠色播放按鈕直接執行程式。
![](PlayButton50)

你可以使用螢幕右上角的「+」按鈕建立更多程式碼檔案。
你可以將一個視窗拖曳到另一個視窗上以將其停靠到另一個視窗下。

你會注意到，一旦你開始輸入，就會彈出一個簡單的程式碼補全視窗。
按 T​​ab 鍵插入程式碼補全。
使用方向鍵瀏覽補全選項。

如果這是你第一次寫程式，請不用擔心。該語言是逐步解鎖的，因此你不會因為功能繁多而不知所措。
它的語法也與 Python 相似，Python 是世界上使用最廣泛的程式語言之一，因此學習它並非完全浪費時間。

如果你已經了解 Python，這也不是問題，你可以快速略過遊戲早期，直接進入更有趣的部分。

目前有兩個可用的無人機指令：

`harvest()`

和

`do_a_flip()`

這些是函式呼叫。你可以將函式視為可執行的命令。使用 `()` 括號執行它。

嘗試在程式碼視窗中輸入以下陳述式，然後按下執行按鈕。

你可以將程式碼視為一系列陳述式。你可以像這樣連續運行多個陳述式：

```
harvest()
do_a_flip()
harvest()
```

## 解鎖
收集牧草可以獲得乾草。乾草可用於在科技樹中解鎖迴圈。使用右上角的按鈕開啟科技樹。
