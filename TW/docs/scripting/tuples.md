# 元組
元組是將多個值組合成單一值的好方法。
要建立元組，只要用逗號分隔值：

```
tuple = 1, 2
```

你也可以將元組再次解包到多個變數中。在下列程式中，元組 `(1,2)` 被解包為兩個變數 `a` 和 `b` 中。

`a, b = 1, 2`

元組可以像串列一樣以索引存取，但它們是不可變的，建立後無法更改。

```
tuple = 1, 2
```

```
print(tuple[1])
```

會印出 `2`

```
tuple[0] = 3
```

與串列不同的是，元組可以當作字典的鍵。

```
d = {(1,2):(4,5)}

print(d[(1,2)])
```

它們也可以用於在函式中傳回多個值。

```
def f():
    return 1, 2

a, b = f()
```
