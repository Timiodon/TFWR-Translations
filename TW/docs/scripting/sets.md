# 集合
集合類似[字典](docs/scripting/dicts.md)，但沒有值，它們只是鍵的無序集合。

它們的建立方式與字典相同，但沒有值。
`set = {North, East, West}`

使用 `set()` 建立空集合。注意，`{}` 會建立空字典。

使用 `set.add(elem)` 新增一個元素到集合。

使用 `set.remove(elem)` 從集合中移除一個元素。

使用 `if elem in set:` 檢查集合是否包含某元素。

使用 `for elem in set:` 來迭代集合中的所有元素。
對於較大的集合，`in` 運算子的效能遠快於在串列上的效能。

和字典一樣，集合是無序的，因此無法保證元素被迭代的順序。

此外，集合中的元素是唯一的，因此新增已存在於集合中的元素不會改變集合。
