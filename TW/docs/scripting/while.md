# while 迴圈
你已解鎖 `while` 迴圈以及值 `True` 和 `False`。只要條件為 `True`，`while` 迴圈就會持續執行迴圈主體。

```
while condition:
	# 執行某些操作
```

不用擔心會建立無限迴圈。執行間的延遲會防止程式當掉。

## 入門指南
也許你已嘗試過連續寫多個 `harvest()`：

```
harvest()
harvest()
harvest()
```

這可以讓你在一次程式執行中採收多次。
但如果想要採收超過三次，重複寫相同程式碼並非良好做法。
解法是使用迴圈。
迴圈允許你重複執行相同的程式碼多次。

`while` 迴圈接受一個條件，該條件為只能處於兩種狀態之一的邏輯值：`True` 或 `False`。
這樣的值稱為布林值。

迴圈會執行迴圈內的程式碼直到條件為 `False`。
`while` 迴圈範例如下：

`
while condition:
	# 迴圈主體
	# ...
`

將 `condition` 替換為布林值，並用你想在迴圈中執行的程式碼取代 `# 迴圈主體` 。

有兩個常數布林值可用。常數是程式執行期間不會改變的值。

永遠為`真`的常數布林值寫作 `True`；永遠為`假`的常數布林值寫作 `False`。
因此你可以寫為：

```
while False:
	do_a_flip()
```

或者

```
while True:
	do_a_flip()
```

第一個永遠不會做翻轉，第二個則會無限次執行翻轉（無限迴圈）。

通常建立無限迴圈並非好主意，因為會讓程式當掉，但在這個遊戲中每次迭代之間有延遲，因此它會讓無人機不斷做翻轉，直到你再次按下執行按鈕手動停止為止。

按 Tab 鍵可加入縮排，按 Shift + Tab（或 Backspace）可移除縮排。

迴圈會重複執行冒號後所有被縮排的陳述式。
縮排區塊之後的陳述式會在迴圈結束後執行。
