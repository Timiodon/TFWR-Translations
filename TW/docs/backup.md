# 載入備份
不幸的是，有時存檔檔案會損壞或遺失一些程式碼檔案。如果遇到這種情況，你可以嘗試載入備份。若是這種情況經常發生，請嘗試關閉 Steam 雲端。

每次儲存遊戲時都會建立一個備份，並且會保留少量備份以備不時之需。
這些備份位於[備份目錄](persistent_data_path/Backup)中。它們是[存檔目錄](persistent_data_path/Saves)中存檔的副本。
載入備份最簡單的方法是將要載入的特定備份資料夾複製到存檔目錄。

存檔是一個包含 `save.json` 檔案和一堆 `.py` 檔案的資料夾。
如果你只遺失了幾個程式碼檔案，或者程式碼檔案仍然存在但 `save.json` 檔案已損壞，你也可以使用備份中的對應檔案替換損壞的部分。
