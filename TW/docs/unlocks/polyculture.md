# 混合種植
你可能已經注意到，有時多種植物同時種植時會有更高的產量。
牧草、灌木、樹木和胡蘿蔔在有合適的伴生植物時會增加產量。每個植物的伴生偏好各不相同且無法預測。幸運的是，可以使用 `get_companion()` 測量無人機下方植物的伴生偏好。它會回傳一個元組，第一個元素為它想要的伴生植物類型，第二個元素為它想要伴生植物的位置。

```
plant_type, (x, y) = get_companion()
```

例如，如果你種下一株灌木然後呼叫 `get_companion()`，它可能會回傳類似 `(Entities.Carrot, (3, 5))`。這表示該灌木希望在位置 `(3, 5)` 有胡蘿蔔。因此如果你在 `(3, 5)` 種上胡蘿蔔，然後採收該灌木，會得到更多木材。胡蘿蔔的生長階段並不影響效果。

植物的伴生偏好可能為 `Entities.Grass`、`Entities.Bush`、`Entities.Tree` 或 `Entities.Carrot`。每株植物會隨機選擇，但永遠不會選擇與自己相同的植物。伴生植物的位置可以是該植物三步內的任何位置，但不會是植物本身的位置。

如果無人機下方沒有具有伴生偏好的植物，`get_companion()` 會回傳 `None`。

在未解鎖混合種植之前，產量倍數為 `5`，每次升級時這個倍數會加倍。
