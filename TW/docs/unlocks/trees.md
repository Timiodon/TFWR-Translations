# 樹木
[樹木](objects/tree)比灌木更適合取得木材。每棵樹木會提供 5 個木材。和灌木一樣，它們可以種植在草地或土壤上。

樹木喜歡保留一些空間，如果將樹木相鄰種植會減慢它們的生長速度。對於位於其東、南、西或北方相鄰格子的每一棵樹木，生長時間都會加倍。所以如果你在每個格子都種植樹木，生長時間會變成 `2 * 2 * 2 * 2 = 16` 倍。

<spoiler=顯示>`%` 運算子在這裡會很有用。請記得 `%` 運算子會回傳除法的餘數。偶數除以 `2` 的餘數為 `0`，奇數除以 `2` 的餘數為 `1`。
因此你可以這樣檢查一個數是否為偶數：

```
def is_even(n):
	return n % 2 == 0
```

如果 n 偶數，這會回傳 `True`，否則回傳 `False`。
</spoiler>
