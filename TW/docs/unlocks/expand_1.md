# 擴張 1
<unlock=for>另請詳閱[擴張 2](docs/unlocks/expand_2.md)

</unlock>你的農場變大了！如果無法移動無人機，這個新的空間就沒什麼用處，因此新增了一個新的函式 `move()` 用來移動無人機。`move()` 需要你指定無人機要移動的方向。新增了四個常數：`North, East, South, West`，分別代表「北方」、「東方」、「南方」、「西方」。

例如，`move(North)` 會將無人機向北移動一格。

如果你移動超出農場邊界，無人機會回到農場的另一側。
下列範例程式會持續將無人機向北移動，並在抵達邊界時繞回起點：

```
while True:
	move(North)
```
