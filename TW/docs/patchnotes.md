## 完整補丁說明
重大變更：
- 由於科技樹重製，舊存檔的升級次數將不再有效。當你打開舊存檔時，遊戲會將所有升級重置到合理的水平。
- 如果你重複使用迷宮，現在必須先重新定位寶藏，然後測量其位置，而不是先測量然後重新定位。

遊戲玩法：
- 為農場地塊加入工具提示，懸停時會顯示各種有用資訊。
- 現在在迷宮中的任何地方使用 `measure()` 都會始終回傳目前寶藏的位置。你不再需要專門測量寶藏。
- 現在迷宮中新增了 `can_move()` 函式。
- 南瓜的尺寸現在從 5x5 擴大到 6x6。
- 地塊乾燥行為略有改變。現在每個地塊每 0.1 秒有 10% 的機率變乾，而不再是每 0.8 秒到 1.2 秒變乾 1%。
- 仙人掌的生長時間現在固定為 1 秒。
- 南瓜現在會留下一個枯萎南瓜來表示它們已經死亡。
- 新增 `pet_the_piggy()`。
- `num_unlocked()` 現在透過「感測」解鎖。

音訊：
- 音訊已完全重製。
- 現在開始遊戲時會有音樂！

視覺效果：
- 科技樹的視覺效果已完全重製。
- 螢幕左上角的物品欄面板現在有漂亮的物品拾取視覺效果。
- 仙人掌如果未正確排序，現在會顯示為棕色，使其更加醒目。
- 植物現在種植後即可立即可見，即使它們尚未生長。

程式碼編輯器：
- 雙擊帶下劃線的識別碼現在會選取整個識別碼。
- 如果關閉「製表符轉空格」選項，縮排空格現在會轉換為製表符。
- 縮放現在更加流暢。
- 程式碼補全器現在可以正確感知導入，並能從其他檔案提供建議。
- 新增了用於編輯文字和調整視窗大小的不同遊標。
- 偵錯器步進現在基於程式碼行，而不是模擬的時間刻。

其他：
- 現在可以像在 Python 中一樣連結比較運算子。
- `get_cost(Entities.Hedge)` 和 `get_cost(Entities.Treasure)` 現在傳回 1x1 迷宮的成本。
- 新增解析度設定。
- 新增關閉在程式碼運行時閃爍醒目標示的設定。
- 程式碼執行得到了進一步優化，使遊戲在程式碼運行時運行更加流暢。
- 新增「切換 HUD」按鍵綁定。

修復：
- 修復一些最常見的程式碼醒目標示在其他視窗可見的問題。
- 耕田等作業不再影響水分乾燥行為。
- 修正非美式鍵盤上的按鍵綁定問題。
- 修正豬豬存錢筒有時會移動到離農場很遠的地方的問題。
- 修復肥料不會感染完全長成的植物的問題。
- 修正 `set_world_size(get_world_size())` 重置無人機位置的問題。
- 修正模擬與恐龍互動時導致大量蘋果被遺漏的問題。
- 修正點擊最小化後再開啟選單時導致錯誤訊息四處移動的錯誤。
- 修正在輸入過程中彈出錯誤訊息的問題。

你可能錯過的舊補丁中的重大更改：
- 其他檔案（遊戲內的視窗）中定義的函式不再被隱式導入。現在你必須像在 Python 中一樣解鎖並使用明確導入語句（請參閱「匯入」升級）。
- `Items.Bones` 已重新命名為 `Items.Bone`，因此所有物品均為單數。
- `Entities.Carrots` 已重新命名為 `Entities.Carrot`，因此所有實體均為單數。
- `Grounds.Turf` 已重新命名為 `Grounds.Grassland`，以便於理解。
- `Items.Water_Tank` 已重新命名為 `Items.Water`，因為水箱補充功能已移除。
- `Unlocks.Benchmark` 已替換為 `Unlocks.Timing`。
- `get_op_count()` 已重新命名為 `get_tick_count()`。
- `set_farm_size()` 已重新命名為 `set_world_size()`，以與 `get_world_size()` 保持一致。
- `get_companion()` 現在傳回 `(entity, (x, y))` 形式的元組，而不是列表。
- `trade()` 已從遊戲中移除。
