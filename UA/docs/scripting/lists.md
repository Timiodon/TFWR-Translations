# Списки
Списки — це зручний спосіб зберігати кілька значень в одній змінній.
Створювати нові списки можна так:

`list = [2, True, Items.Hay]`

Тепер список містить значення `2`, `True` та `Items.Hay`.
Також список може бути порожнім:

`empty_list = []`

Отримати елемент списку можна за його індексом. Індекс дорівнює `0` для першого елемента, `1` для другого, `2` для третього тощо.

Цей код садить моркву:

`list = [Entities.Tree, Entities.Carrot, Entities.Pumpkin]
plant(list[1])`

Ви можете перебрати список за допомогою циклу for. Наступний приклад обчислює суму всіх елементів списку:

`list = [4, 7, 2, 5]
sum = 0
for number in list:
	sum += number`
`sum` is now `18`

Наступні методи списку дозволяють додавати та видаляти елементи:

`list.append(elem)` додає елемент в кінець списку:

`list = [2, 6, 12]
list.append(7)`

`list` тепер дорівнює `[2, 6, 12, 7]`

`list.remove(elem)` видаляє перше входження елемента зі списку:

`list = [1, 2, 4, 2]
list.remove(2)`

`list` тепер дорівнює `[1, 4, 2]`

`list.insert(index, elem)` вставляє елемент за вказаним індексом:

`list = [Entities.Tree, Items.Hay]
list.insert(1, Items.Wood)`

`list` тепер дорівнює `[Entities.Tree, Items.Wood, Items.Hay]`

`list.pop(index)` видаляє елемент за вказаним індексом.
Якщо індекс не вказано, видаляється останній елемент:

`list = [3, 5, 8, 25]
list.pop()`

`list` тепер дорівнює `[3, 5, 8]`

`list.pop(1)`

`list` тепер дорівнює `[3, 8]`

Функція `len()` повертає довжину списку:

`list = [3, 2, 1]
x = len(list)`

`x` тепер дорівнює `3`

Списки мають семантику посилання. Це означає, що при присвоєнні списку іншій змінній копія не створюється — обидві змінні посилаються на один об'єкт списку.
Якщо дві змінні посилаються на один список, зміни в ньому будуть видимі для обох:

`a = [1,2]
b = a
b.pop()`

`a` і `b` тепер дорівнюють `[1]`
