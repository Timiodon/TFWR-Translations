# Змінні
Змінні можна уявляти як іменовані контейнери, які зберігають значення.
Оператор `=` використовується для визначення змінної та збереження в ній значення:

`variable_name = value`

Ліва частина оператора — це ім'я змінної. Ви можете назвати її як завгодно.
Права частина — це вираз, результат якого буде збережено в змінній.

Оголосити змінну `a` і зберегти в ній значення `5`:

`a = 5`

Оголосити змінну `b` і зберегти в ній результат виклику `can_harvest()`:

`b = can_harvest()`

Не плутайте оператор `=` з оператором `==`. 
Оператор `==` перевіряє, чи два значення рівні, і повертає `True` або `False`.
Оператор `=` присвоює значення з правої частини імені з лівої частини.

Після присвоєння змінну можна використовувати в коді для отримання її значення:

`a = 5
for i in range(a):
	do_a_flip()`

Цей цикл виконається 5 разів, тому що `a` дорівнює `5`.
`i` в циклі `for` також є змінною — на кожній ітерації їй автоматично присвоюється поточне значення з послідовності. (Її не обов'язково називати `i`, ви можете використати будь-яке допустиме ім'я змінної.)

Те саме можна зробити за допомогою циклу `while`:

`a = 5
i = 0
while i < a:
	do_a_flip()
	i = i + 1`

Цей код робить те саме, що і попередній приклад. Але тут ми збільшуємо `i` вручну.
Зверніть увагу, що для збільшення `i` ми встановлюємо її значення, рівним власному значенню плюс `1`. Зміна значення змінної на основі її попереднього значення є дуже поширеним. 
Його можна скоротити, використовуючи дані оператори: `+=, -=, *=, /=, %=`

`i = i + 1` еквівалентно `i += 1`
`a = a / 3` еквівалентно `a /= 3`
