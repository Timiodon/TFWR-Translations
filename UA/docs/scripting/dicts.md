# Словники
Словники — це структура даних, яка дозволяє зіставляти ключі зі значеннями, так само як у справжньому словнику зіставляються слова з їхніми значеннями. Завдяки цьому пошук відбувається дуже швидко.

Словник можна створити так:
`right_of = {North:East, East:South, South:West, West:North}`

Вираз перед двокрапкою — це ключ, а вираз після двокрапки — значення, з яким пов'язаний цей ключ.
Наведений вище словник зіставляє кожен напрямок із напрямком праворуч від нього.

Ось ще один приклад, який зіставляє позицію дрона з об'єктом, над яким він знаходиться:
`x, y = get_pos_x(), get_pos_y()
entity_dict = {(x,y):get_entity_type()}`

Доступ до значення за ключем подібний до доступу до елемента списку:
`value = dict[key]`

Приклад:
`orientation = right_of[South]`
Цей код встановлює значення `orientation` на `West`.

Додати нову пару ключ-значення до словника можна так:
`dict[key] = value`

Приклад:
`entity_dict[(get_pos_x(), get_pos_y())] = get_entity_type()`
Цей код оновлює об'єкт, збережений для поточної позиції.

Ключі є унікальними, тож додавання ключа, який уже існує в словнику, перезапише попереднє значення.

Use `dict.pop(key)` to remove a key-value pair from `dict`.

`key in dict` набуває значення `True`, якщо `key` є ключем у `dict` і `False` в іншому випадку.
Тому можна використовувати `if key in dict:`, щоб перевірити, чи містить `dict` цей ключ.

Помістивши словник у цикл for, можна перебрати всі ключі:
`for key in dict:
	value = dict[key]`

Немає жодних гарантій щодо порядку, в якому ключі будуть перебиратися.
