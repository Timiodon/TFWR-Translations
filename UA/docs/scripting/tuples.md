# Кортежі
Кортежі є зручним способом комбінувати кілька значень в одне.
Для створення кортежу достатньо просто розділити значення комами:

`tuple = 1, 2`

Ви також можете розпакувати кортеж в кілька змінних. У наведеному нижче коді кортеж `(1,2)` розпаковується в змінні `a` і `b`:

`a, b = 1, 2`

Кортежі можна індексувати так само, як списки, але вони є незмінними і не можуть бути змінені після створення:

`tuple = 1, 2`

`print(tuple[1])`

виводить `2`

`tuple[0] = 3`

спричинить помилку
<unlock=dicts>
На відміну від списків, кортежі можна використовувати як ключі в словниках.

`d = {(1,2):(4,5)}

виводить(d[(1,2)])`
`prints` (4,5)</unlock>

Також вони можуть бути корисними для повернення кількох значень із функції.

`def f():
    return 1, 2

a, b = f()`
