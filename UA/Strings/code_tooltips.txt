виконується за 1 тік.

приклад:
`abs(-69)`
@code_tooltip_random = `random()`
Генерує випадкове число між 0 (включно) та 1 (виключно).

повертає випадкове число.

виконується за `1` тік.

приклад:
`def random_elem(list):
	index = random() * len(list) // 1
	return list[index]`
@code_tooltip_append = `list.append(element)` 
Додає `element` до кінця `list`.

повертає `None`

виконується за `1` тік.

приклад:
`list = []
list.append(1)`
@code_tooltip_add = `set.add(element)` 
Додає `element` до `set`.

повертає `None`

виконується за `element size` тіків.

приклад:
`set = {0}
set.add(1)`
@code_tooltip_remove = `collection.remove(element)` 
Видаляє перше входження `element` з `collection`.

повертає `None`

виконується за `element size` тіків у множині, #comparisons + #shifts тіків у списку.

приклад:
`list = [True, False, None]
list.remove(False)`
@code_tooltip_pop = `collection.pop()` 
Видаляє останній елемент зі списку або вказаний елемент зі словника.
`list.pop(i)` видаляє елемент за індексом `i` з `list`.

повертає видалений елемент

виконується за `key size` тіків у словнику, `len(list) - i + 1` тіків у списку.

приклад:
`list = [True, False, None]
list.pop(0)`
@code_tooltip_insert = `list.insert(i, element)` 
Вставляє `element` в `list` за індексом `i`.

повертає `None`

виконується за `len(list) - i + 1` тіків.

приклад:
`list = [1,2]
list.insert(0, 0)`

@code_tooltip_Grounds = Містить усі типи ґрунту. Можна ітерувати за допомогою циклу `for`.
@code_tooltip_Unlocks = Містить усі розблокування та покращення з меню досліджень. Можна ітерувати за допомогою циклу `for`.
@code_tooltip_Hats = Містить усі типи капелюхів. Можна ітерувати за допомогою циклу `for`.
@code_tooltip_Leaderboards = Містить усі категорії таблиці лідерів. Можна ітерувати за допомогою циклу `for`.
@code_tooltip_for = Цикл, що ітерується по всіх елементах послідовності. У деяких мовах програмування він називається циклом "foreach".
@code_tooltip_while = Цил, що виконується, доки умова істинна.
@code_tooltip_def = Визначає функцію.
@code_tooltip_True = Булеве значення, яке завжди істинне.
@code_tooltip_False = Булеве значення, яке завжди хибне.
@code_tooltip_if = Виконує код, якщо умова є `True`.
@code_tooltip_else = Виконує код, якщо попередня умова `if` була `False`.
@code_tooltip_elif = Робить те саме, що:
`else:
    if condition:`
@code_tooltip_None = Значення, що представляє відсутність значення.
@code_tooltip_continue = Негайно переходить до наступної ітерації циклу. Якщо цикли вкладені, впливає завжди на найвнутрішніший цикл.
@code_tooltip_break = Виходить з циклу та продовжує виконання після циклу. Якщо цикли вкладені, впливає завжди на найвнутрішніший цикл.
@code_tooltip_North = Напрямок вгору на екрані. Якщо тільки ви не перевернули екран.
@code_tooltip_East = Напрямок вправо на екрані. Якщо тільки ви не перевернули екран.
@code_tooltip_South = Напрямок вниз на екрані. Якщо тільки ви не перевернули екран.
@code_tooltip_West = Напрямок вліво на екрані. Якщо тільки ви не перевернули екран.
@code_tooltip_not = `not True` дорівнює `False` і `not False` дорівнює `True`.
@code_tooltip_and = Обчислює перший операнд. Якщо він хибний (`False`, `0` та порожні колекції), відразу повертає його значення (короктке замикання), інакше обчислює та повертає другий операнд.
@code_tooltip_or = Обчислює перший операнд. Якщо він істинний (будь-що, крім `False`, `0` та порожніх колекцій), відразу повертає його значення (коротке замикання), інакше обчислює та повертає другий операнд.
@code_tooltip_return = Використовується для повернення значення з функції.
@code_tooltip_pass = Нічого не робить. Може бути корисним, так як порожні блоки коду заборонені.
