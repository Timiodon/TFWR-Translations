# Зовнішній редактор
Вбудованого ігрового текстового редактора зазвичай достатньо для гри, але він, звісно, не може конкурувати з більш просунутими редакторами, такими як Visual Studio Code.

Гра зберігає всі файли коду як файли `.py`, тому ви можете редагувати їх у будь-яких редакторах для Python. 
Зауважте, що це зроблено лише для зручності. Мова в грі насправді не є Python, але вона достатньо схожа на неї, щоб функції автодоповнення (IntelliSense) працювали належним чином.
Ви можете знайти ці файли в [папці збережень](persistent_data_path/Saves).

Кожне збереження також містить файл `__builtins__.py`, який містить вбудовані визначення Python, що відповідають ігровим функціям для роботи IntelliSense.
VS Code може виявляти `__builtins__.py` автоматично, але деякі інші редактори можуть працювати лише якщо ви додасте рядок `from __builtins__ import *`.

Щоб бачити зовнішні зміни в грі без необхідності перезавантажувати збереження, ви повинні увімкнути опцію "Стеження за файлами" (File Watcher). Якщо ви створюєте або видаляєте файли ззовні, вам все одно доведеться перезавантажити збереження, щоб побачити зміни.

## Використання VS Code
Visual Studio Code — це рекомендований редактор коду для використання з "The Farmer Was Replaced".

Ви можете встановити його [тут](https://code.visualstudio.com/download).

Після завантаження встановіть розширення Python у самому VS Code.

Як тільки ви це зробите, відкрийте [папку](persistent_data_path/Saves), де зберігаються ваші файли `.py`, у VS Code. Переконайтеся, що ви відкрили саме всю папку, а не лише окремі файли, інакше файл `__builtins__.py` не працюватиме.

У грі переконайтеся, що опція "Стеження за файлами" (File Watcher) увімкнена. Тепер щоразу, коли ви зберігаєте файл у VS Code, зміни автоматично з'являтимуться в грі.

Ось і все! Тепер ви можете писати свій код у професійному редакторі!
