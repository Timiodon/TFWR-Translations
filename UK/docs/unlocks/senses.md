# Датчики (Senses)
Тепер дрон може бачити!

Функції `get_pos_x()` та `get_pos_y()` повертають поточні координати x та y дрона. У початковій позиції вони обидві дорівнюють `0`. Координата x збільшується на `1` з кожною клітинкою у напрямку `East` (схід), а координата y збільшується на `1` з кожною клітинкою у напрямку `North` (північ).

`num_items(item)` повертає кількість одиниць певного предмета, яка у вас є.
Наприклад, `num_items(Items.Hay)` поверне кількість сіна.

`get_entity_type()` та `get_ground_type()` повертають тип об'єкта або землі, що знаходиться під дроном.

Зробити сальто, якщо ви знаходитесь над кущем:
`if get_entity_type() == Entities.Bush:`
`	do_a_flip()`

Ключове слово `None` також тепер розблоковано! `None` — це значення, яке вказує на відсутність значення.
Наприклад, функція, яка не має інструкції `return`, насправді поверне `None`.

`get_entity_type()` повертає `None`, якщо під дроном немає жодного об'єкта.


Якщо ви хочете дізнатися, скільки покращень певного типу ви розблокували, використовуйте функцію `num_unlocked(unlock)`.

Наприклад, `num_unlocked(Unlocks.Speed)` поверне кількість розблокованих вами покращень швидкості.

`num_unlocked(Unlocks.Senses)` поверне `1`, якщо датчики розблоковані, і `0`, якщо ні.

Ви також можете використовувати `num_unlocked()` для предметів (`Items`), об'єктів (`Entities`) або типів зхемлі (`Grounds`). Функція поверне `1`, якщо вони розблоковані, інакше — `0`.

Будьте уважні: `num_unlocked(Unlocks.Carrots)` поверне кількість разів, скільки було придбано це розблокування.
`num_unlocked(Items.Carrot)` поверне `0` або `1`. (Аналогічно для інших рослин)
