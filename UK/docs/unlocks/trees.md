# Дерева (Trees)
[Дерева](objects/tree) — це кращий спосіб отримати деревину, ніж кущі. Вони дають по 5 одиниць деревини кожне. Як і кущі, їх можна висаджувати на пасовищі або ґрунті.

Дерева люблять простір, тому посадка їх впритул одне до одного сповільнить їхній ріст. Час росту подвоюється за кожне дерево, яке знаходиться на сусідній клітинці безпосередньо на півночі, сході, заході або півдні. Отже, якщо ви посадите дерева на кожній клітинці, вони ростимуть у `2*2*2*2 = 16` разів довше.

<spoiler=show> Оператор `%` може бути корисним тут. Пам’ятайте, що оператор `%` повертає остачу від ділення. Парні числа при діленні на `2` мають остачу `0`, а непарні — `1`.
Отже, ви можете перевірити, чи є число парним, таким чином:

`def is_even(n):`
`	return n % 2 == 0`

Функція повертатиме `True`, якщо n — парне, і `False`, якщо ні.
</spoiler>
