# Вартості (Costs)
Будь-які вартості можуть бути представлені у вигляді словника, який зіставляє предмети (items) з їхньою кількістю.

Функція `get_cost()` повертає такий словник. Вона показує вартість рослини або розблокування.

`get_cost(Entities.Pumpkin)`
повертає `{Items.Carrot:1}`

Для розблокувань можна передати необов'язковий другий аргумент — рівень розблокування, вартість якого ви хочете дізнатися. За замовчуванням це поточний рівень розблокування.

`get_cost(Unlocks.Loops, 0)`
повертає `{Items.Hay:5}`

Для розблокувань, які вже досягли максимального рівня, `get_cost()` поверне `None`.

Це можна використовувати ось так:
`cost = get_cost(something)
for item in cost:
	amount_of_this_item_needed = cost[item]`
