# Автоматичне розблокування (Auto Unlocks)
Щоб повністю автоматизувати гру, ви можете використовувати функцію `unlock()` для автоматичного розблокування нових можливостей.
Наприклад, ви можете використати `unlock(Unlocks.Speed)` та `unlock(Unlocks.Expand)`, щоб розблокувати функції швидкості та розширення території.

Щоб визначити вартість розблокування, просто скористайтеся функцією `get_cost()`, як ви б це зробили для рослини або предмета.
Приклад:
`get_cost(Unlocks.Loops)`
повертає `{Items.Hay:5}`
