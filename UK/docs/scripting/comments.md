# Коментарі (Comments)
Коментарі — це частини коду, які ігноруються під час виконання програми.
Коментар можна додати за допомогою символу `#`. Усе, що написано в тому ж рядку після `#`, вважається коментарем і не буде виконуватися.

`# це коментар`

Це корисно для додавання нотаток до коду, а також для тимчасового вимкнення частин коду без їхнього видалення.

У цій грі будь-який коментар у рядку безпосередньо перед визначенням функції стане частиною інформаційної підказки, яка з’являється при наведенні курсору на назву функції.

`# Ця функція нічого не робить.

def f():
    pass`
