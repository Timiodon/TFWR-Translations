# Кортежі (Tuples)
Кортежі — це чудовий спосіб об'єднати кілька значень в одне.
Щоб створити кортеж, просто розділіть значення комами:

`tuple = 1, 2`

Ви також можете знову розпакувати їх у кілька змінних. У наведеному нижче коді кортеж `(1,2)` розпаковується у дві змінні `a` та `b`.

`a, b = 1, 2`

Кортежі можна індексувати як списки, але вони є незмінними (immutable) і не можуть бути змінені після створення.

`tuple = 1, 2`

`print(tuple[1])`
надрукує `2`

`tuple[0] = 3`
викличе помилку
<unlock=dicts>
На відміну від списків, кортежі можна використовувати як ключі у словниках.

`d = {(1,2):(4,5)}

print(d[(1,2)])`
надрукує `(4,5)`</unlock>

Вони також можуть бути корисними для повернення кількох значень із функції.

`def f():
    return 1, 2

a, b = f()`
