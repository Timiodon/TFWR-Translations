# Множини (Sets)
Множини схожі на [словники](docs/scripting/dicts.md), але без значень. Це просто невпорядкований набір ключів. 

Вони створюються так само, як словники, але без значень:
`set = {North, East, West}`

Використовуйте `set()`, щоб створити порожню множину. Зауважте, що `{}` створює порожній словник.

Використовуйте `set.add(elem)`, щоб додати новий елемент до множини.

Використовуйте `set.remove(elem)`, щоб видалити елемент із множини.

Використовуйте `if elem in set:`, щоб перевірити, чи містить множина певний елемент.

Використовуйте `for elem in set:`, щоб перебрати всі елементи множини.
Для великих множин оператор `in` працює набагато швидше, ніж для списків.

Як і словники, множини є невпорядкованими, тому немає жодних гарантій щодо порядку, у якому перебиратимуться елементи.

Також елементи в множинах є унікальними, тому додавання елемента, який уже є в множині, нічого в ній не змінить.
