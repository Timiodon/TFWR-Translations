# Змінні (Variables)
Змінні можна уявити як іменовані контейнери, які можуть зберігати значення.
Оператор `=` використовується для оголошення змінної та збереження в ній значення.

`variable_name = value`

Ліва частина оператора — це ім'я змінної. Ви можете дати їй будь-яке ім'я.
Права частина — це вираз, результат якого буде збережено у змінній.

Оголосити змінну з іменем `a` і зберегти в ній значення `5`:
`a = 5`
Оголосити змінну з іменем `b` і зберегти в ній результат виконання функції `can_harvest()`:
`b = can_harvest()`

Не плутайте оператор `=` з оператором `==`.
Оператор `==` перевіряє, чи є два значення рівними, і повертає `True` або `False`.
Оператор `=` присвоює значення яке розміщене праворуч від оператора до імені ліворуч.

Після того, як змінній було присвоєно значення, ви можете використовувати її в коді, щоб отримувати значення, яке раніше туди записали:

`a = 5
for i in range(a):
	do_a_flip()`

Наведений вище цикл виконається 5 разів, оскільки `a` дорівнює `5`.
У циклі `for` `i` — це також змінна, якій автоматично присвоюється поточне значення послідовності на кожній ітерації циклу. (Її не обов'язково називати `i`, ви можете дати їй будь-яке допустиме ім'я змінної).

Змінні також дозволяють зробити те саме за допомогою циклу `while`:

`a = 5
i = 0
while i < a:
	do_a_flip()
	i = i + 1`

Це робить те саме, що і цикл `for` вище. Нам лише потрібно збільшувати `i` вручну.
Зверніть увагу: щоб збільшити `i`, ми встановлюємо її значення рівним її попередньому значенню плюс `1`. Зміна значення змінної на основі її попереднього значення — це дуже поширена операція.
Її можна скоротити за допомогою таких операторів: `+=, -=, *=, /=, %=`

`i = i + 1` — це те саме, що й `i += 1`
`a = a / 3` — це те саме, що й `a /= 3`
