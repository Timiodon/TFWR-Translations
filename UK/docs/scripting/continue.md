# Continue
continue дозволяє зупинити поточну ітерацію циклу та перейти до наступної ітерації поточного циклу.

`for i in range(10):
	continue
	print("це ніколи не буде надруковано")`

Це запустить усі `10` ітерацій циклу, але інструкція `print` після `continue` завжди буде пропускатися.

Це також працює з циклами `while`.

`while True:
	if not can_harvest():
		continue
	
		harvest()`

Цей код викликає `harvest()` тільки тоді, коли `can_harvest()` повертає `True`.
Це має такий самий ефект, як і:

`while True:
	if can_harvest():
		harvest()`

У вкладених циклах `continue` завжди впливає на найближчий (внутрішній) цикл.

`for i in range(10):
	for j in range(10):
		print("це буде надруковано 100 разів")
		continue
		print("це ніколи не буде надруковано")
	print("це буде надруковано 10 разів")`
