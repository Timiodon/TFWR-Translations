# První program
## Textový editor
Veškeré programování probíhá v oknech s kódem. Každé okno s kódem odpovídá textovému souboru s kódem.
Soubor můžete přejmenovat kliknutím na jeho jméno v horní části okna.

Kód lze upravovat jako v běžném textovém editoru, dokud neběží.
Program můžete spustit přímo stisknutím zeleného tlačítka pro přehrání v okně s kódem.
![](PlayButton50)

Nové soubory s kódem vytvoříte pomocí tlačítka "+" vpravo nahoře.
Okno můžete připnout k jinému oknu tažením na něj.

Jakmile začnete psát, objeví se jednoduché doplňování kódu.
Stiskněte `Tab` pro vložení vybrané položky doplňování.
Použijte šipky pro pohyb v nabídkách doplňování.

Nemějte obavy, pokud programujete poprvé. Jazyk se odemyká postupně, takže nebudete zahlceni vším najednou.
Syntax je podobná Pythonu, což je jeden z nejpoužívanějších programovacích jazyků, takže se to hodí i mimo hru.

Pokud už Python znáte, není to problém — rychle přeskočíte počáteční části a dostanete se k zajímavějším věcem.

V současnosti jsou k dispozici dvě příkazy pro drony.

`harvest()`

a

`do_a_flip()`

Jedná se o volání funkcí. Funkci si můžete představit jako příkaz, který lze vykonat — volá se pomocí závorek `()`.

Zkuste napsat tyto příkazy do okna s kódem a stisknout tlačítko pro spuštění.

Kód lze vnímat jako posloupnost příkazů. Můžete spouštět více příkazů za sebou takto:

`harvest()
do_a_flip()
harvest()`

## Odemykání
Sbíráním trávy získáte seno. Seno lze použít k odemykání smyček v menu odemykání. Otevřete menu odemykání tlačítkem v pravém horním rohu.
