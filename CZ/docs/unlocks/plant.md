# Sázení
Tráva je fajn, protože roste automaticky. Všechny ostatní rostliny musí být zasazeny pomocí funkce `plant()`. Jediná rostlina, kterou můžete právě teď zasadit, je keř.
Typ rostliny, kterou chcete zasadit, můžete předat funkci takto:

`plant(Entities.Bush)`

Tím se zasadí keř pod drona.

Zavolejte `clear()` pro resetování farmy na samou trávu a resetování pozice drona.

Zdá se, že pokud pěstujete na farmě více než jeden typ rostliny současně, můžete někdy získat vyšší výnos. Budete muset vyzkoumat polykulturu, abyste se dozvěděli více.
